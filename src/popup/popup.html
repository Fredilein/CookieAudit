CookieAudit

  • When you start a scan, all cookies are deleted
  • Make sure to close all other tabs before starting a scan
  • During the scan, navigate around the site
  • Don't open any other website while scanning
  • This is still an early version of the extension! It might not work properly
Info: You can use this extension to check if a website you are interested in sets cookies that you have not allowed, for example, advertising cookies. To do that, navigate to the website in question and click the Start button.
If you just want to test this extension, you can navigate to a popular website (for example, www.computerbild.de) and, when a consent banner appears, select "Allow All". This way you can see what happens when a website sets cookies that are used for advertising purposes, for example.
